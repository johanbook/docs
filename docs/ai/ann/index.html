Basics

An Artificial Neural Network (ANN) is a computational graph, that is a network of nodes where every connection can have different strengths. In mathematical terms a single node yyy be expressed as

y=ϕ(∑iωixi)y = \phi\left(\sum_i \omega_i x_i\right)y=ϕ(i∑​ωi​xi​)

where ϕ\phiϕ is the activation function of the node, xix_ixi​ its inputs and ωi\omega_iωi​ the weights. The network is trained by optimizing yyy via ω\omegaω using an optimizer.

Bayesian Neural Network​

A Bayesian neural network is an ANN that uses a probability distribution for the weights instead of point estimates. This will yield networks that can be sampled but will also increase the number of parameters that needs to be trained.
